Windows Logo Key Keyboard Shortcuts

By Xah Lee. Date: 2011-06-21

This page lists all Windows logo key shortcuts that works in Windows 7. Windows Vista supports vast majority of these keys.

In general, any key combo with the Windows logo key ❖ Window is a global keyboard shortcut. Key combo that start with the ▤ Menu key (aka “Application key”) is supposed to be application specific, but that never caught on.

Introduced in Windows XP

  • 【❖ Window+B】 Select the first icon in the Notification Area.
  • 【❖ Window+D】 Show the desktop, or restore hidden programs when pressed a second time.
  • 【❖ Window+E】 Open Windows Explorer (view folders).
  • 【❖ Window+F】 Open Windows Search.
  • 【❖ Window+Ctrl+F】 Open Search for Computers. Requires Active Directory Domain Services.
  • 【❖ Window+F1】Open Windows Help.
  • 【❖ Window+L】 Lock desktop or switch users.
  • 【❖ Window+M】 Minimize all windows.
  • 【❖ Window+Shift+M】 Restore windows that were minimized with 【❖ Window+M】.
  • 【❖ Window+R】 Open the Run dialog.
  • 【❖ Window+U】 Run the Utility Manager, known as “Ease of Access” in later versions of Windows.
  • 【❖ Window+Pause】 Open System Properties.

Introduced in Windows XP Media Center Edition

  • 【❖ Window+Alt+Enter】 Start Windows Media Center interface.

Introduced in Windows Vista

  • 【❖ Window+G】 Cycle Windows gadgets.
  • 【❖ Window+Spacebar】 to show the Windows Sidebar. Replaced with Aero Peek in Windows 7, as the Windows Sidebar was discontinued.
  • 【❖ Window+X】 Enter the Windows Mobility Center. Works only if portable computer features are installed.
  • 【❖ Window+Tab】 Switch windows with Aero Flip 3D. (desktop composition must be turned on)
  • 【❖ Window+Ctrl+Tab】 Switch windows using sticky Aero Flip 3D.
  • 【❖ Window+1】, 【❖ Window+2】, … 【❖ Window+0】 Launch app on the Quicklaunch bar. In Windows 7, these runs the apps pinned to the taskbar.

Introduced in Windows 7

  • 【❖ Window+Home】 Toggle hide all other windows.
  • 【❖ Window+Spacebar】 View desktop. (activate Aero Peek).
  • 【❖ Window+P】 Choose a presentation display mode (set screen to clone on 2nd monitor, etc.).
  • 【❖ Window+↑】 Maximize current window.
  • 【❖ Window+↓】 Restore current window's default window size.
  • 【❖ Window+Shift+↑】 Maximize current window vertically.
  • 【❖ Window+Shift+↓】 Restore current window vertically, or minimize window.
  • 【❖ Window+←】 or 【❖ Window+→】 Align the window to the side of the screen, maximizing it vertically.
  • 【❖ Window+Shift+←】 or 【❖ Window+Shift+→】 Move the window to the next or previous monitor, if multiple monitors are used
  • 【❖ Window+T】 Iterate through items on the taskbar.
  • 【❖ Window++】 Zoom-in screen. 【❖ Window+-】 Zoom-out screen. 【❖ Window+Escape】 Cancel magnification.
  • 【❖ Window+1】, 【❖ Window+2】, … 【❖ Window+0】 to run or activate the corresponding programs in the taskbar.
  • 【❖ Window+Shift+1】 … start a new instance of the app on taskbar.
